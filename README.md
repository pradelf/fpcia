# Francis Pradel Intelligence Artifcielle (fpcia) Library

This is a standalone Python library which contains code for  working with various AI library such as :
- scikit-learn
- pytorch

## Setup

Python >=3.10

- clone the repository

```bash
git clone 
```

- install the library

```bash
pip install .
```

- to run tests:

```bash
pip install pytest
```

```bash
pytest
```
